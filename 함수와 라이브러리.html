함수 — 내장 함수와 사용자 정의 함수
함수란?
특정 기능을 수행하는 명령문을 하나로 묶은 코드 단위입니다. 같은 작업을 반복할 때 함수로 만들면 코드를 더 간결하게 작성할 수 있습니다.
함수의 기본 구조
def 함수 정의
매개변수 함수에 전달되는 값
return 결과값 반환
내장 함수
print(), input(), int()
float(), str(), len()
range(), type(), abs()
사용자 정의 함수
def add(a, b):
return a + b
print(add(3, 5))
함수 호출
함수명()
함수명(값)
변수 = 함수명(값)

예제: 인수 없고 반환값 없음

def hello(): print("Hello world") hello()

예제: 인수 있고 반환값 있음

def add(a, b): return a + b result = add(3, 5) print(result)

예제: 환영 메시지 함수

def greet(name): print("환영합니다.") print(name, "님 입장") print("공지사항 확인하세요.") greet("홍길동") greet("김민지")

모듈과 라이브러리
모듈
함수, 변수, 클래스를 모아 놓은 하나의 파이썬 파일입니다. import 모듈명으로 불러와 사용합니다.
라이브러리
여러 모듈을 모아 놓은 집합입니다. 파이썬 기본 제공 라이브러리와 설치가 필요한 외부 라이브러리로 나뉩니다.

모듈 사용 형식

import 모듈명 import 모듈명 as 별명 모듈명.함수() 별명.함수()
math

random
난수 생성: randint(), sample()
별명 사용

rd.randint(1, 10)

import math print(math.sqrt(121)) print(math.gcd(15, 18)) print(math.pi)

외부 라이브러리 — pandas, numpy, matplotlib

외부 라이브러리는 표 데이터 처리, 배열 계산, 데이터 시각화처럼 기본 기능만으로 구현하기 어려운 작업을 쉽게 처리하도록 도와줍니다.

pandas

표(DataFrame) 형태의 데이터를 분석하고 조작합니다.

import pandas as pd df = pd.DataFrame({"Name":["Alice","Bob"], "Age":[25,30]}) print(df)
numpy

배열과 수치 계산을 효율적으로 처리합니다.

import numpy as np arr = np.array([1,2,3,4,5]) print(np.mean(arr)) print(np.max(arr)) print(np.sum(arr))
matplotlib

그래프와 차트를 그려 데이터를 시각화합니다.

import matplotlib.pyplot as plt x = [1,2,3,4,5] y = [20,30,65,87,93] plt.scatter(x, y) plt.show()

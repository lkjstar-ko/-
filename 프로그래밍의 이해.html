변수와 자료형
변수란?
프로그램에서 값을 저장하는 기억 공간. = 을 이용해 저장하고 변경할 수 있다.
형식
변수명 = 저장할 값
(상수, 변수, 수식 등)

자료형 4가지

int 정수
예: 10, -3, 0
소수점 없는 정수
float 실수
예: 2.5, 3.14
소수점 있는 실수
str 문자열
예: "A", "안녕"
따옴표로 감싼 문자
bool 불리언
예: True, False
참/거짓 논리값

코드 예시

# 변수 저장 및 변경 a = 10 b, c = 2.5, "A" print(a, b, c) # 10 2.5 A a = a * 2 print(a, b, c) # 20 2.5 A

표준입출력과 파일입출력

입력하고, 처리하고, 출력하기

이 단원은 파이썬 프로그램이 사용자와 데이터를 주고받는 방법을 익히는 부분입니다. 먼저 input()과 print()로 화면 입출력을 연습한 뒤, open(), read(), write()로 파일에 저장하고 읽는 흐름을 배웁니다.

1. 표준 입력키보드로 값을 받기
2. 표준 출력화면에 결과 보여주기
3. 파일 입출력파일에 저장하고 읽기
표준입출력 핵심
input()입력

사용자가 키보드로 입력한 값을 문자열로 가져옵니다. 숫자로 계산하려면 int() 또는 float()로 바꿉니다.

name = input("이름: ") age = int(input("나이: "))
print()출력

변수, 문자열, 계산 결과를 화면에 출력합니다. 여러 값을 쉼표로 연결하면 공백으로 구분되어 출력됩니다.

print("안녕하세요") print(name, "은", age, "살입니다.")
파일입출력 흐름
1단계 파일 열기open()
→
2단계 읽기·쓰기read() / write()
→
3단계 파일 닫기close()
"r" 읽기
기존 파일의 내용을 읽습니다.
"w" 쓰기
파일 내용을 새로 저장합니다.
"a" 추가
파일 끝에 내용을 덧붙입니다.
파일 쓰기저장
f = open("memo.txt", "w") f.write("안녕하세요") f.close()
파일 읽기확인
f = open("memo.txt", "r") data = f.read() print(data) f.close()
실습 순서
  1. 예제 버튼을 눌러 코드를 불러옵니다.
  2. input()이 있으면 표준 입력값을 줄 단위로 적습니다.
  3. 실행 버튼을 누르고 결과를 확인합니다.

컴퓨터 이미지 표현

흑백 이미지 (2차원)
밝기값(0~255)만으로 구성된 2차원 배열
0 = 검정, 255 = 흰색
컬러 이미지 (3차원)
빨강(R), 초록(G), 파랑(B) 3채널
각 채널 0~255 밝기값

이미지 시각화 (matplotlib)

import matplotlib.pyplot as plt # 흑백 이미지 (2차원) gray = [[255,0],[0,255]] plt.imshow(gray, cmap='gray') # 컬러 이미지 (3차원 RGB) color = [[[255,0,0],[0,255,0]]] # 빨강, 초록 plt.imshow(color) plt.show()

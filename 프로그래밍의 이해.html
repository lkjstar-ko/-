변수와 자료형
변수란?
프로그램에서 값을 저장하는 기억 공간. = 을 이용해 저장하고 변경할 수 있다.
형식
변수명 = 저장할 값
(상수, 변수, 수식 등)

자료형 4가지

int 정수
예: 10, -3, 0
소수점 없는 정수
float 실수
예: 2.5, 3.14
소수점 있는 실수
str 문자열
예: "A", "안녕"
따옴표로 감싼 문자
bool 불리언
예: True, False
참/거짓 논리값

코드 예시

# 변수 저장 및 변경 a = 10 b, c = 2.5, "A" print(a, b, c) # 10 2.5 A a = a * 2 print(a, b, c) # 20 2.5 A

표준입출력과 파일입출력

① 표준입출력 — input() / print()

input()

키보드로 데이터를 입력받는 함수. 입력값은 문자열(str)로 처리됨.

변수명 = input() 변수명 = input("안내 문구") # 숫자로 쓰려면 int() 변환 필요 n = int(input("숫자: "))
print()

처리 결과를 모니터로 출력하는 함수. 콤마로 여러 값 출력 가능.

print(값) print("문자열", 변수명) print(a, b, c) # 공백 구분

파일 읽기

f = open("파일.txt", "r") data = f.read() print(data) f.close()

파일 쓰기

f = open("파일.txt", "w") f.write("저장할 내용") f.close()

다차원 데이터 구조

컴퓨터가 처리하는 자료는 다양한 형태로 존재하므로 자료의 특성에 맞는 데이터 구조를 선택해야 합니다.

1차원

하나의 행으로 구성된 선형 데이터 구조

[80, 100, 90]
2차원

행과 열로 구성된 표 형태

[[80,100,90],
[93,85,70],
[55,72,75]]
3차원

행·열·높이로 구성된 입체 구조

이미지(RGB)
R·G·B 3채널

컴퓨터 이미지 표현

흑백 이미지 (2차원)
밝기값(0~255)만으로 구성된 2차원 배열
0 = 검정, 255 = 흰색
컬러 이미지 (3차원)
빨강(R), 초록(G), 파랑(B) 3채널
각 채널 0~255 밝기값

이미지 시각화 (matplotlib)

import matplotlib.pyplot as plt # 흑백 이미지 (2차원) gray = [[255,0],[0,255]] plt.imshow(gray, cmap='gray') # 컬러 이미지 (3차원 RGB) color = [[[255,0,0],[0,255,0]]] # 빨강, 초록 plt.imshow(color) plt.show()

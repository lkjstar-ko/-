자료 수집 및 핵심 요소 추출 (20) 문제 정의, 알고리즘 구상, 코드 구조 설계 등 개발 과정 전반을 충실히 수행하고, 관련 기술 요소를 적극적으로 반영함. 설계와 개발 활동을 수행했으나 과정 일부(문제 정의·알고리즘·구조 설계 중 일부)가 미흡함. 제한적인 설계나 개발 활동만 이루어졌으며 핵심 요소 반영이 부족함. 설계 및 개발 활동을 수행하지 못함.
프로그래밍 보고서 제작하기 (20) 문제 정의, 알고리즘 구상, 코드 구조 설계 과정을 포함하여 보고서를 체계적으로 작성하고, 서론–본론–결론 형식을 충실히 갖추어 독창적으로 완성함. 문제 정의, 알고리즘 구상, 코드 구조 설계 중 일부가 미흡하거나 서론–본론–결론 형식은 갖추었으나 내용이 부족함. 프로그래밍 설계 보고서를 작성하였으나 핵심 요소(문제 정의, 알고리즘 구상, 코드 구조 설계) 대부분이 누락되었거나 불완전함. 프로그래밍 설계 보고서를 작성하지 못함.
프로그래밍 보고서 발표하기 (20) 자신이 설계한 프로그래밍 아이디어와 과정(문제 정의 → 알고리즘 구상 → 코드 구조 설계)을 체계적으로 발표하고, 발표 자료를 충실히 공유함. 자신이 설계한 프로그래밍 아이디어와 과정을 발표하였으나 일부 과정 설명이 미흡하거나 발표 자료 공유가 불완전함. 자신이 설계한 프로그래밍 아이디어나 과정 중 일부만 발표하거나 자료 공유가 부족함. 프로그래밍 설계 발표를 하지 못하였으며, 자료도 공유하지 못함.

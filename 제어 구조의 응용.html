조건문 — if / elif / else
구문역할실행 조건
if첫 조건 확인조건식이 True일 때 실행
elif추가 조건 확인앞 조건이 False이고 이 조건이 True일 때
else나머지 처리앞 모든 조건이 False일 때 실행

if문 형식

if 조건식 : 명령문1 # 참일 때 실행 명령문2 # 조건과 상관없이 실행

if-elif-else 형식

if 조건식1 : 명령문1 elif 조건식2 : 명령문2 else : 명령문3

예제: 점수 등급

score = 85 if score >= 90 : print("A") elif score >= 80 : print("B") else : print("C 이하")

반복문 — for / while / break / continue

for 문

for 변수 in range(횟수) : 반복할 명령문

while 문

while 조건식 : 반복할 명령문 # 조건이 True인 동안 반복

예제: 1~100 합계

t = 0 for a in range(1, 101) : t += a print(t) # 5050

제어 구조 프로그래밍 — 중첩 & 조합

조건문과 반복문을 조합하면 더 복잡한 실생활 문제를 해결할 수 있습니다.

예제: 구구단 전체

for dan in range(2, 10) : for i in range(1, 10) : print(dan, "x", i, "=", dan*i)

예제: 조건 + 반복

for n in range(1, 11) : if n % 2 == 0 : print(n, "짝수")

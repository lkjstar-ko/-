과제 개요 및 지침

개요 및 제출 요건

프로젝트 기본 정보

  • 주제: 자유 선정 (아래 예시 참고)
  • 형식: 구글 슬라이드 1장 (제공된 양식 사용)

필수 제출물

  1. PDF 변환 파일: (파일명: 학번이름.pdf)
  2. 코랩(Colab) 파일: (파일명: 학번 이름.ipynb)
  3. 사용한 데이터셋: (.csv, .xlsx 등 원본 파일)
주제 예시 (자유 선정)
  • 보건/건강: 나이·BMI로 비만 여부 예측
  • 환경: 기상 데이터로 날씨 분류 (맑음/흐림/비)
  • 교육: 공부 시간·수면 시간으로 합격/불합격 예측
  • 일상생활: 키·몸무게로 성별 분류
  • 기타: Kaggle, UCI 등 교육용 데이터셋 활용 가능
데이터 준비, 모델 학습 및 비교

데이터 준비 및 분석 요구사항

  • 입력 속성: 2~4가지 특징 선택
  • 분류 클래스: 2~3가지 그룹으로 분류

필수 전처리 과정:

  • 결측치 처리
  • 필요 시 정규화/표준화
  • 데이터 분할 (훈련/검증/테스트)

모델 학습 및 비교 요구사항

  • 최소 3가지 분류 모델 적용 (예: KNN, 로지스틱 회귀, SVM, 결정 트리 등)
  • 각 모델의 정확도 계산 및 비교
  • 정확도 외에도 장단점, 활용 가능성 분석 포함
필수 스크린샷 및 성찰(Reflection) 문항

반드시 포함해야 할 스크린샷 (Colab 결과)

  • 데이터 시각화 결과
  • 데이터 전처리/분할 과정 코드 및 결과
  • 세 가지 모델의 정확도 비교 결과 (표 또는 그래프)

성찰(Reflection) 문항 (슬라이드에 작성)

  1. 어떤 모델이 가장 적합했는가? 이유는 무엇인가?
  2. 프로젝트 과정에서 어려웠던 점과 해결 방법은 무엇인가?
  3. 이 프로젝트가 실제 생활에 어떻게 활용될 수 있을까?
제출 및 평가 기준 (총 100점)

제출 기한 및 제출처 (필수 준수)

  • 기한: 수행평가 마지막일 수업 종료시
  • 제출처: 구글클래스룸 과제방

평가 기준 (총 100점)

1. 프로젝트 완성도 (20점)
  • 모든 항목이 빠짐없이 작성되어 있는가
  • 각 항목의 내용이 명확하고 논리적인가
  • 시각화 자료와 실행 결과가 적절히 포함되어 있는가
2. 데이터 분석의 정확성 (40점)
  • 데이터 전처리가 적절히 수행되었는가
  • 세 가지 모델의 정확도가 올바르게 계산되었는가
  • 데이터 시각화에 대한 해석이 타당한가
  • 전처리-분석-시각화-해석 과정이 논리적으로 일관되게 연결되었는가
3. 결과 분석의 충실성 (20점)
  • 모델별 성능 비교가 명확히 제시되었는가
  • 새로운 데이터 예측 결과가 포함되어 있는가
  • 프로젝트 성찰이 구체적으로 작성되었는가
  • 시각화 자료를 활용하여 성능 비교 및 해석이 타당하게 이루어졌는가
  • 분석 결과의 실제 응용 가능성이 제시되었는가
4. 제출 요건 준수 (20점)
  • 데이터세트, 코랩 파일, 슬라이드 파일을 모두 제출하였는가
  • 파일명 규칙을 준수하였는가
  • 제출 형식 및 제출 기한(수업 종료 전)을 준수하였는가

② 논리 연산자 (Logical Operators)

and — 둘 다 참일 때만 True
True and True → True
True and False → False
False and False → False
score >= 60 and attend >= 80
or — 하나라도 참이면 True
True or False → True
False or True → True
False or False → False
hobby == '독서' or hobby == '게임'
not — 참↔거짓 반전
not True → False
not False → True
not (age < 13)

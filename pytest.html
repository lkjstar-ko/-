제출 안내

필수 입력 항목

  • 이름: 실명을 입력하세요.
  • 학번: 숫자 5자리로 입력하세요.
  • 답안: 모든 문제의 빈칸을 완성하세요.

학번 입력 형식

학번은 학년 + 반 + 번호 형식입니다.

1학년 1반 1번 → 10101

2학년 3반 14번 → 20314

답안 작성 시 주의사항

  • 빈칸 _____ 부분을 수정하세요.
  • 괄호, 따옴표, 들여쓰기를 함부로 삭제하지 마세요.
  • 제출 전 모든 문제를 작성했는지 확인하세요.

제출 후 저장되는 내용

  • 이름
  • 학번
  • 제출 시간
  • 작성 코드
  • PDF 파일
  • Python 파일(.py)

문제 1. 변수

name 변수에 자신의 이름을 저장하고 출력되도록 완성하세요.

문제 2. 자료형

정수형 변수 age를 만들고 자신의 나이를 저장하세요.

문제 3. 문자열

city 변수에 "제주"를 저장하고 출력하세요.

문제 4. 덧셈 연산자

두 수의 합이 출력되도록 완성하세요.

문제 5. 나눗셈 연산자

100을 4로 나눈 결과가 출력되도록 완성하세요.

문제 6. 문자열 연결

first와 last를 연결하여 fullname이 출력되도록 완성하세요.
